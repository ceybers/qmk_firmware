# Useful Excel Leader Mnemonics

## Rows
R S - Row select
R D - Row delete
R F - Row (auto) fit
R C - Row clear

## Columns
C S - Col select
C D - Col delete
C F - Col (auto) fit
C C - Col clear

## Filters
F V - Filter by value (current cell)
F N - Filter non blank
F B - Filter blanks
F R - Filter reapply
F C - Clear filter on column
F R - Clear all filters (Filter Remove)
F D - Filter dropdown
F / - Filter custom (? key) (dropdown + ff -> custom filter dialog)

## Selections
S V - Select visible
S A - Select all
S E - Select from here until end (Ctrl+Shift+End)
S H - Select from here until home

## Paste
V ? - normal paste?
V V - Values only
V F - Formatting only
V A - Paste Append
